# LegacyInsights
This project aims to provide a comprehensive platform for analyzing and visualising artifact data from various countries and time periods. 
Built upon a foundation of data exploration and visualisation, this tool goes beyond mere analysis by incorporating advanced features such as interactive plotting, statistical analysis, and machine learning models.
